Instructions:
1) Download Processing 3 for your system (https://processing.org/download/)
2) Start Processing (Select More Info > Run Anyway if you get a Windows Defender warning)
3) Use File > Open... to open assignment1_shell/assignment1_shell.pde.
4) Click the "Play" button in Processing to run the example.  You should see text information about the L-System appear by pressing the 1-6 keys.  
5) Find the method called drawLSystem() has been defined.  This method currently just prints the text information about the current L-System.  This method is where you should create your implementation for drawing an L-System.
